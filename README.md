# Connect – Real-Time Chat Application

A real-time chat application built using **Node.js, Express, and Socket.IO** that allows users to join chat rooms and communicate instantly with a modern responsive interface.

## Features

* Real-time messaging using WebSockets
* Multiple chat rooms
* Online users list in each room
* Profanity filter for messages
* Responsive UI for desktop and mobile

## Tech Stack

* **Backend:** Node.js, Express
* **Real-time Communication:** Socket.IO
* **Frontend:** HTML, CSS, JavaScript

## Installation

```bash
git clone https://github.com/<your-username>/connect.git
cd connect-chat-app
npm install
npm start
```
